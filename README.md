# 📍 Preenchimento Automático de CEP

Projeto desenvolvido com HTML, CSS e JavaScript para realizar a busca automática de endereços a partir de um CEP informado pelo usuário.

## 🚀 Funcionalidades

* Consulta de CEP em tempo real
* Preenchimento automático de endereço
* Exibição de rua, bairro, cidade e estado
* Validação de CEP
* Interface simples e intuitiva

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* API ViaCEP

## 📂 Estrutura do Projeto

```text
/
├── index.html

```

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/preenchimento-cep.git
```

2. Acesse a pasta do projeto:

```bash
cd preenchimento-cep
```

3. Abra o arquivo `index.html` em seu navegador.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

* Consumo de APIs REST
* Requisições assíncronas com Fetch API
* Manipulação do DOM
* Tratamento de erros
* Validação de formulários
* Integração com serviços externos

## 🎯 Objetivo

O objetivo deste projeto é demonstrar como utilizar APIs para automatizar o preenchimento de formulários, melhorando a experiência do usuário e reduzindo erros de digitação.
